## Wildzekat/bot-resources
This repository stores **public** resources for some of my bots.

This includes:
- Terms pages
- Privacy policies
- Usage guides

Each bot receives it's own folder.
I don't expect anyone to read this file, users usually land in one of the separate pages via a button.

In any case you still landed here, maybe in search for a place to report bot / app issues to me:
Please just contact me directly. This usually makes it easier for me to locate the issue and works best.

In case you dont want, or cannot, contact me directly, please leave an issue using one of the two templates.

Content may appear in english or german. I will not append translations to each file, so long as my target group is solely german.

### Bot-Ressourcen
Dieses Repository lagert **öffentliche** Ressourcen für meine Bots.

Dazu gehören:
- Nutzungsbedingungen
- Datenschutzerklärungen
- Endnutzeranweisungen

Jedem Bot wird ein eigener Ordner zugewiesen.
Ich erwarte eigentlich nicht, dass hier je jemand erscheint, Nutzer sollten lediglich zu einzelnen Seiten über Buttons geführt werden.

Falls ihr trotzdem hier seid, zum Beispiel auf der Suche nach einem Ort um Fehler zu melden:
Schreibt mich bitte einfach direkt an. Das vereinfacht mir einiges.

Falls ihr mich nicht erreichen könnt oder wollt, lasst ein Issue mithilfe einer der Vorlagen hier. Die Vorlagen sind auf __Englisch__.

Inhalte können sowohl in Englisch als auch in Deutsch hier erscheinen. Ich übersetze nur Inhalte, die beide Zielgruppen ansprechen.
